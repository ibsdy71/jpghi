### 放大倍数怎么选

官方有个实用建议：放大倍数不是越大越好。倍数越高，AI 需要"脑补"的细节越多，处理时间也更长。通常 2x 需要 1-2 分钟，4x 要 5-6 分钟，6x 要将近 10 分钟，而 16x 可能需要 30-60 分钟。

对于大多数日常需求（手机照片提升清晰度、给老图翻新），4x 通常是效果和等待时间的甜蜜点。
